# EcommerceAdminPortal

An Angular-based admin dashboard project built following the [Build Admin Dashboard in Angular](https://youtu.be/uI7u9YpzjSg?si=p6vv-CvP39xPrXmV) tutorial on YouTube by "CodingShiksha". This project demonstrates how to build a modern, responsive, and modular admin interface for e-commerce platforms using Angular 20 and ngx-admin UI libraries.

## 🔧 Tech Stack

- **Angular CLI** v20.0.1
- **Angular** v20
- **ngx-charts** for data visualization
- **ng2-search-filter** for filtering data
- **Bootstrap** (via CDN or Angular integration)
- **FontAwesome** for icons
- **Reactive Forms** for form handling
